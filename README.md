# 📚 Amazon Bestsellers Analysis (2009–2019)

End-to-end data analysis project on 10 years of Amazon
bestselling books — from messy raw data to interactive dashboard.

## 🖼 Dashboard Preview
!overview_dashboard.png

## 🔄 Project Workflow
Raw Data (Excel) → Cleaning (Python ) → Analysis → Power BI

## 🔍 Key Insights
- 348 unique books appeared on bestseller lists over 10 years
- Non-Fiction slightly dominates: 54.6% vs Fiction 45.4%
- Avg user rating is consistently high: 4.28/5.0
- 4 books appeared 11 times — most repeated bestsellers
- Avg price: $26.31 | Highest reviewed: 345K reviews
